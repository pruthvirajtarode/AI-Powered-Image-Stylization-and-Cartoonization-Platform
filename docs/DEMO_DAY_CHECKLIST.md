### During Issues

#### If Login Fails
- [ ] Stay calm - don't panic
- [ ] Say: "Let me try with our backup account..."
- [ ] Use demo2@toonify.ai
- [ ] Keep talking (don't go silent)
- [ ] Joke: "The AI got a little too creative..."

#### If Image Won't Upload  
- [ ] Check file size
- [ ] Refresh browser
- [ ] Use pre-processed image (show screenshot as backup)
- [ ] Continue - don't dwell on it
- [ ] Say: "Let me show you the processed version..."

#### If Processing Is Slow
- [ ] Don't wait silently - talk through features while processing
- [ ] Say: "While this is processing, let me explain..."
- [ ] Have backup image ready to show
- [ ] Never say "this is usually faster"

#### If Camera Doesn't Work
- [ ] Switch to upload method
- [ ] Say: "Let me show you the drag-and-drop instead..."
- [ ] Don't apologize - keep the energy up
- [ ] Show different feature

#### If Internet Drops
- [ ] Have screenshots of demo ready
- [ ] Say: "Let me show you this captured version..."
- [ ] Try refresh/reload
- [ ] Use 4G mobile hotspot if available
- [ ] If data-heavy issue, go to slides only

#### If Someone Asks Hard Question
- [ ] Pause (2 seconds to think)
- [ ] Answer honestly
- [ ] If unsure: "That's a great question - let me get back to you on that"
- [ ] Don't BS
- [ ] Have business cards ready for follow-up
